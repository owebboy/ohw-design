About

Created by Oliver Pope, ohw is a file uploader that makes it super easy to share files. All you need is a twitter account to get started.

Why?

ohw was created to fix an annoyance I've had as a student. Their was no easy way to upload html to the web for homework assignments and projects.

How?

Using nodejs and advanced coding skillz I was able to build a simple web application for uploading files to the internet, easily.
